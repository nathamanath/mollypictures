Mollypictures

Place Holder Images Featuring Molly the Bunny Rabbit as a Service (PHIFMtBRaaS)

Usage:

set image source to:

/IMAGE_WIDTHxIMAGE_HEIGHT.FORMAT

Where:

  • `IMAGE_WIDTH` is desired image with in pixels (<=1920)
  • `IMAGE_HEIGHT` is desired image height in pixels (<=1080)
  • `FORMAT` is one of `.jpg`, `.jpeg`, `.gif` or `.png`

or grayscale:

/g/IMAGE_WIDTHxIMAGE_HEIGHT.FORMAT
